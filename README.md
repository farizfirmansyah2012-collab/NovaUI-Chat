# NovaChat — AI Chat Dashboard

NovaChat adalah template AI Chat Dashboard modern dengan tampilan premium bergaya SaaS. Dirancang menggunakan HTML, CSS, dan JavaScript murni tanpa framework sehingga ringan, responsif, dan mudah dikustomisasi.

Template ini menghadirkan pengalaman chat AI yang modern lengkap dengan dukungan multi-model, panel workspace, riwayat percakapan, serta integrasi API yang siap digunakan.

---

## Preview

NovaChat menghadirkan desain bertema dark modern dengan efek glassmorphism, animasi halus, dan layout profesional yang cocok untuk berbagai proyek AI.

---

## Features

- 💬 Modern AI Chat Interface
- 🤖 Multi AI Model Support
  - GPT-5
  - Claude Sonnet 5
  - Gemini 3 Pro
  - Llama 4
- 🔑 API Key Management
- 📂 Chat History Sidebar
- ⚙️ AI Workspace Panel
- 🌡️ Temperature Control
- 📊 Token Usage Indicator
- ✨ Typing Animation
- 📱 Fully Responsive
- 🌙 Premium Dark UI
- ⚡ Lightweight (No Framework)
- 🎨 Glassmorphism Design
- 🖥️ Desktop & Mobile Optimized
